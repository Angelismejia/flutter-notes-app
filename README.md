# 📝 Flutter Notes App

![Flutter](https://img.shields.io/badge/Flutter-3.0-blue?logo=flutter)
![Dart](https://img.shields.io/badge/Dart-3.0-blue?logo=dart)
![License](https://img.shields.io/badge/License-MIT-green)

Aplicación móvil de notas desarrollada con Flutter y Hive.

## ✨ Características
- Crear, editar y eliminar notas
- Búsqueda y filtros por categoría
- Colores personalizados por nota
- Favoritos con estrella
- Dark mode
- Contador de notas
- Ordenar por fecha o alfabéticamente

## 🛠️ Tecnologías
- Flutter / Dart
- Hive (base de datos local)
- Material Design 3

## 🚀 Instalación
```bash
git clone https://github.com/Angelismejia/flutter-notes-app.git
cd flutter-notes-app
flutter pub get
flutter run
```
